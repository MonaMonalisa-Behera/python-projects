# Movie Recommendation System

This is a simple Python project that recommends movies to the user.

## Technologies Used
- Python

## How it Works
The program stores a list of movies.
When the user selects a movie, the system recommends other movies from the list.

## Example

Input:
User selects "Avatar"

Output:
Recommended movies:
Titanic
Avengers
Inception
